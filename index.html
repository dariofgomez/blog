Conceptos básicos en la terminal

Navegar dentro de la terminal:

Estos comandos sirven para moverse entre las carpetas del computador a través de la terminal:
* pwd: indica la carpeta actual.
* ls: mostrar los archivos dentro de la carpeta actual.
* ls -al: mostrar todos los archivos (incluidos los ocultos) en forma de lista.
* cd +nombre de la carpeta: entrar a una carpeta dentro de la carpeta actual.
* cd .. : ir a la carpeta anterior.












git:

Este comando sirve para verificar que se haya instalado Git en el PC.




git init:

Este comando sirve para inicializar Git dentro de la carpeta del proyecto a trabajar.




git status:

Este comando sirve para verificar el estado de los archivos del proyecto, si fueron modificados, si estan añadidos, o si estan listos para guardar.
